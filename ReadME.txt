# To-Do List Application

## Overview
This is a simple console-based To-Do List application developed in Python. Users can add tasks, view saved tasks, and manage their daily activities through an interactive menu.

## Features
- Add new tasks
- View all tasks
- Menu-driven interface
- Input validation

## Technologies Used
- Python
- Lists
- Loops
- Conditional Statements

## How to Run
1. Install Python.
2. Download the source code.
3. Run:

python todo_list.py

## Learning Outcomes
- Working with Python Lists
- Using Loops and Conditions
- Handling User Input
- Building Menu-Driven Applications
